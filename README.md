# 🛒 Ecommerce Website UI

A modern and visually structured e-commerce website UI built using HTML, CSS, and JavaScript. This project focuses on layout design, UI components, and interactive features like dropdowns and countdown timers.

---

## 🚀 Features

- Multi-level navigation bar with icons
- Search bar with category filter
- Custom dropdowns (Country & Currency selector)
- Hero section with categories, banner, and user panel
- Deals & Offers section with live countdown timer
- Product showcase with discount badges
- New arrivals section (Home & Electronics)
- Supplier inquiry form
- Recommended items grid
- Clean and structured UI design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Responsive Layout)
- JavaScript (DOM manipulation, events, countdown logic)
- Font Awesome (icons)

- Updates every second using JavaScript
